## Run Locally

Clone the project

```bash
  git clone https://github.com/v1st78/World-Ranks
```

Go to the project directory

```bash
  cd World-Ranks
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
